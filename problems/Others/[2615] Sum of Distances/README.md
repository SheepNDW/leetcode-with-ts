## 2615. Sum of Distances

給定整數陣列 `nums`，對每個索引 `i` 計算**所有與 `nums[i]` 值相同的其他元素**與位置 `i` 的索引距離總和，回傳結果陣列。

**範例：**

```
Input:  nums = [1, 3, 1, 1, 2]
Output: arr  = [5, 0, 3, 4, 0]
```

`arr[0] = |0-2| + |0-3| = 5`，`arr[2] = |2-0| + |2-3| = 3`，以此類推。


### 思路

#### 暴力解（O(n²)）

對每個 `i`，遍歷所有相同值的索引逐一求距離和，n 大時 TLE。

#### 前綴和滑動（O(n)）

先將相同值的索引收集成群組，對群組內的索引由左往右滑動，利用前一個位置的結果遞推：

移動前，以群組第一個索引 `indices[0]` 計算初始距離總和。接著每往右移動一步到第 `i` 個索引時，左邊 `i` 個元素距離各增加 `diff`，右邊 `m - i` 個元素距離各減少 `diff`，其中 `diff = indices[i] - indices[i-1]`：

```
sum += diff * i - diff * (m - i)
```
