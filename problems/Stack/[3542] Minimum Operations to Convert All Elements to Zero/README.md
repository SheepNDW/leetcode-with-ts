## 3542. Minimum Operations to Convert All Elements to Zero

我們可以觀察到，如果同時將多個相同的極小值設為 0，可以減少操作次數。然後若在兩個相同數字之間存在其他較小的數字，則這些數字無法在同一次操作中被設為 0。

我們可以維護一個單調遞增的 stack 來追蹤目前非零元素的遞增序列。對於每一個元素 `num`，我們執行以下步驟：

* 如果 stack 頂端的元素大於 `num`，那麼根據剛才的觀察，我們需要將 stack 頂端的元素彈出。
* 如果 stack 頂端的元素等於 `0`，我們就跳過它，因為不需要再對它進行操作。
* 如果 stack 頂端的元素小於 `num`，說明我們需要一個新的操作來將 `num` 設為 0，因此我們將 `num` 推入 stack，並將操作次數加 1。

最後回傳總的操作次數即可。
