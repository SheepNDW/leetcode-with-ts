## 2141. Maximum Running Time of N Computers

這題給了一個整數陣列 `batteries`，表示有多少電池，每個電池有多少電量，還有一個整數 `n`，表示有多少台電腦。我們要找出這 `n` 台電腦能夠同時運行的最長時間。

我們假設 n 台電腦所能同時運作的最長時間為 `T`，那麼每一台電腦在這段時間內都需要消耗 `T` 的電量。總共 `n` 台電腦需要的電量為 `n * T`。因為電池可以任意分配給各個電腦使用，所以策略很簡單，只要把所有電池的電量加總起來，檢查是否能夠滿足 `n * T` 即可。但需要注意的是，任一個電池的使用時間不能超過 `T`，因為我們只會讓電腦運行 `T` 時間，所以在計算可用電量時，對每個電池取 `min(battery, T)`。

那接下來我們只要用常規度二分搜尋法來找出最大的 `T` 即可，不斷去猜測 `T`，如果 `T` 是可行解，就代表我們可以嘗試更大的 `T`，反之則嘗試更小的 `T`。收斂解果即為答案。
