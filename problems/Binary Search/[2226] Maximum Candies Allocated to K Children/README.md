## 2226. Maximum Candies Allocated to K Children

非常明顯的二分搜值的題目。

令每個孩子能分得的糖果數目為 `x`，如果 `x` 很大，那麼我們可以構造出符合條件（每堆 `x` 個糖果）的堆數就會變少，極有可能最終不夠 `k` 堆。反之，如果 `x` 很小，那麼我們可以構造出符合條件的堆數就會變多，甚至超過 `k` 堆。因此，我們可以對 `x` 進行二分搜值，嘗試不同的 `x` 值，來逼近最大的符合條件的 `x` 值。

對於給定的 `x`，將每堆糖果數除以 `x`，就是該堆能夠拆出的糖果堆數。我們只需檢查符合條件的總堆數是否大於等於 `k` 即可。

註：此題和 1891. Cutting Ribbons 類似。
