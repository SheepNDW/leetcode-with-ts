## 840. Magic Squares In Grid

掃過每個 3×3 子矩陣，檢查是否為魔方陣。魔方陣必須滿足：

* 9 個格子內的數字必須是 1~9 且各出現一次
* 每一列、每一行、以及兩條對角線的總和都相同

由於必須使用 1~9 各一次，因此子矩陣總和固定為 `1+...+9 = 45`。又因為 3 列的總和都相同，所以每一列（也就是 magic sum）必須是 `45 / 3 = 15`，因此每一行、每一列、兩條對角線都必須等於 15。

接著準備一個 helper function 檢查某個 3×3 是否為魔方陣：先用 `Set` 記錄 9 個數字，若出現重複或不在 1~9 範圍內就直接回傳 0；否則分別計算 3 列、3 行、2 條對角線的總和是否都等於 15，全部通過則回傳 1。

整體時間複雜度為 `O(m*n)`
