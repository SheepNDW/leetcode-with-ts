## 3212. Count Submatrices With Equal Frequency of X and Y

給一個二維矩陣，裡面只有 `'X'`、`'Y'`、`'.'` 三種字元。計算有多少個子矩陣同時滿足以下三個條件：

- **條件一：包含左上角元素**
  子矩陣的左上角必須是整個矩陣的 `[0][0]` 位置，也就是每個合法子矩陣都是從原點 `(0, 0)` 延伸到某個 `(i, j)` 的矩形區域。

- **條件二：至少包含一個 `'X'`**
  子矩陣內 `'X'` 的數量必須 `≥ 1`。

- **條件三：`'X'` 與 `'Y'` 的數量相等**
  子矩陣內 `'X'` 的總數必須等於 `'Y'` 的總數。

### 思路

與 3070 相同，核心工具是**二維前綴和**，只是這次需要同時維護兩個前綴和矩陣：

- `prefixX[i][j]`：從 `(0, 0)` 到 `(i, j)` 的子矩陣中，`'X'` 的數量
- `prefixY[i][j]`：從 `(0, 0)` 到 `(i, j)` 的子矩陣中，`'Y'` 的數量

兩者都套用標準的二維前綴和遞推公式，差別只在最後加上的值：

```ts
prefixX[i][j] = prefixX[i-1][j] + prefixX[i][j-1] - prefixX[i-1][j-1] + (grid[i][j] === 'X' ? 1 : 0)
prefixY[i][j] = prefixY[i-1][j] + prefixY[i][j-1] - prefixY[i-1][j-1] + (grid[i][j] === 'Y' ? 1 : 0)
```

建構完成後，對每個位置 `(i, j)` 檢查：

```
prefixX[i][j] > 0 && prefixX[i][j] === prefixY[i][j]
```

若成立，計數器加一。
