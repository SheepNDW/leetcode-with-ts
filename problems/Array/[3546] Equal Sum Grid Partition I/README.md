## 3546. Equal Sum Grid Partition I

給一個 m × n 的正整數矩陣，判斷是否能對矩陣進行**一次水平切割**或**一次垂直切割**，使得：

- 切出的兩個區域都非空
- 兩個區域的元素總和相等

回傳 `true` 或 `false`。

### 解題思路

首先計算矩陣所有元素的總和 `totalSum`。若 `totalSum` 為奇數，兩半不可能相等，直接回傳 `false`。

若 `totalSum` 為偶數，問題轉化為：**能不能找到一條切割線，使得切割線某一側的總和恰好等於 `totalSum / 2`？**

預先計算每一列的總和 `sumRow` 與每一欄的總和 `sumCol`，接著用**一維前綴和**分別掃描：

- 水平方向：逐列累加 `sumRow`，若某個前綴累加值 `=== target` 且後面還有剩餘列，回傳 `true`
- 垂直方向：逐欄累加 `sumCol`，若某個前綴累加值 `=== target` 且後面還有剩餘欄，回傳 `true`

「後面還有剩餘」這個條件是為了確保切出的兩個區域都非空，因此迴圈只跑到 `m - 1` 和 `n - 1`。
