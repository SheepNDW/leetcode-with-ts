## 1975. Maximum Matrix Sum

題目給了一個 `n * n` 矩陣 `matrix`，你可以對任意一對「相鄰（上下左右）」的元素同時乘以 `-1`，操作次數不限。目標是讓矩陣所有元素總和最大，並回傳該最大值。

要最大化總和，直覺上希望所有元素最後都變成正數，因此先計算所有元素的絕對值總和 `total = Σ |matrix[i][j]|`，這會是理想上限。

矩陣是連通的，負號可以被搬運，因此只剩奇偶性限制答案。但並不是任何情況都能把所有元素翻成正數，關鍵在於一個不變量：「矩陣中負數的個數的奇偶性」。

- 每次操作會同時翻轉兩個元素的正負號，因此「負數個數」只可能 `±2` 或 `0`，所以負數個數的奇偶性（奇/偶）永遠不變。

那麼根據負數個數的奇偶性，可以分成兩種情況：

- 若負數個數 `negCount` 是偶數：
  - 可以透過適當操作，將所有元素都變成正數。
  - 因此最大總和就是 `total`。

- 若負數個數 `negCount` 是奇數：
  - 無法將所有元素都變成正數，最後至少會有一個元素是負數。
  - 為了最大化總和，應該讓剩下來的那個負數的絕對值（`minAbs`）最小化。

最後的最大總和計算方式就會是：

- 若 `negCount` 是偶數，最大總和為 `total`。
- 若 `negCount` 是奇數，最大總和為 `total - 2 * minAbs`。

整體的時間複雜度為 `O(n^2)`（掃過整個矩陣一次），空間複雜度為 `O(1)`。
