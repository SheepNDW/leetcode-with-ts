## 3623. Count Number of Trapezoids I

給定一個由整數座標點組成的陣列 `points` ，請你計算並回傳由這些點所能組成的梯形數量。且題目要求梯形的兩條平行邊必須與 x 軸平行。

我們可以理解為，上底是一條「同一 y 座標、高度較高」的水平線段，下底是一條「同一 y 座標、高度較低」的水平線段。兩條底邊的 y 不同，自然就能組成梯形且平行於 x 軸。因此我們不需要去管左右兩邊是直的還是斜的，只要確保上底和下底是水平線且 y 座標不同即可。

我們可以把問題拆解開來：

1. 同一水平線段上的點，可以兩兩組合成一條底邊。
     - 假設某個 y 上有 k 個點，則可以組成的線段數量為 `C(k, 2) = k * (k - 1) / 2`。
2. 梯形如何組成？
     - 在某個 y1 上，假設可以組成 A 條線段；在另一個 y2 上，假設可以組成 B 條線段。
     - 那麼這兩條線段就能組成 A * B 個梯形。（任選一條上底和一條下底）
3. 總體答案的公式
     - 假設把每個高度 y 所對應的水平邊數量定義為 `lines[i]`，則：
     - $ans = \sum_{i < j} lines[i] * lines[j]$

如何有效率地計算 $ans$ 呢？

假設 `lines = [a0, a1, a2, ..., an]`，我們想計算：

$$\sum_{i < j} a_i * a_j$$

可以利用 prefix sum 的技巧，維護一個 `presum`，代表目前為止的 `a` 的總和。

```ts
let ans = 0;
let presum = 0;

for (const a of lines) {
  ans += presum * a;
  presum += a;
}
```

最後這題需要注意的地方是，使用一般 number 型別可能會 overflow，因此需要使用 BigInt 來處理。
