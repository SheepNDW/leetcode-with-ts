## 2654. Minimum Number of Operations to Make All Array Elements Equal to 1

只有兩數互質時，才能透過有限次的操作弄出一個 1，只要出現了一個 1，那它與相鄰元素結合，之後每個回合都能再產生一個 1，而且這是最高效的方式。

所以我們也可以得到，如果所有元素的最大公因數不是 1，那就不可能產生 1，直接回傳 -1。

再來，如果陣列中已經存在 k 個 1，那麼我們只需 `n - k` 次操作就能把所有元素變成 1。

最後，我們需要找出最短的子陣列，使得該子陣列的最大公因數為 1，於是我們就可以將題目簡化為「求一個最短區間，使得該區間的 gcd 為 1」。我們直接遍歷所有的 subarray，找到最短區間長度 `len`，那麼我們需要的操作次數就是 `len - 1 + n - 1`，其中 `len - 1` 是將該區間縮減為 1 所需的操作次數，`n - 1` 是將剩下的元素變成 1 所需的操作次數。
