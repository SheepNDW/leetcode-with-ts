## 3197. Find the Minimum Area to Cover All Ones II

將一個矩陣分成三個互不相交的子矩形，只有如下六種形式：

```
        1.        2.      3.
                    
        ┌－┐      ┌┐┌┐    ┌－┐
        └－┘      └┘└┘    └－┘
        ┌－┐      ┌－┐    ┌┐┌┐
        └－┘      └－┘    └┘└┘
        ┌－┐
        └－┘
        
        2.       5.      6.
        ┌┐┌┐┌┐   ┌ ┐┌┐    ┌┐┌ ┐
        └┘└┘└┘   │ │└┘    └┘│ │
                 │ │┌┐    ┌┐│ │
                 └ ┘└┘    └┘└ ┘
```

對於每種形式，只有兩條分割線。我們可以用 `O(MN)` 的時間遍歷分割線的位置，就可以確定三個子矩陣的邊界。對於每一個子矩陣，我們再遍歷其中的元素，確定包含所有元素1的最小矩陣即可
