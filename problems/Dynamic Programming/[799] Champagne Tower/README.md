## 799. Champagne Tower

這題模擬香檳塔的倒酒過程。從最頂端倒入 `poured` 杯香檳，每個杯子最多裝 1 杯，溢出的部分會平均流向下一層左右兩個杯子，問第 `query_row` 層第 `query_glass` 個杯子裝了多少酒。

直接用 DP 模擬整個流動過程即可。令 `dp[i][j]` 代表流入第 `i` 層第 `j` 個杯子的總量（注意不是最終持有量，而是流入量，可能超過 1）。

初始狀態為 `dp[0][0] = poured`，所有酒從頂端倒入。

對於每個杯子 `(i, j)`，如果流入量超過 1，溢出的部分 `dp[i][j] - 1` 會平均分配給下一層的兩個杯子：

```ts
if (dp[i][j] > 1) {
  dp[i + 1][j] += (dp[i][j] - 1) / 2;
  dp[i + 1][j + 1] += (dp[i][j] - 1) / 2;
}
```

最後回傳 `Math.min(1, dp[query_row][query_glass])`，因為杯子最多只能裝 1 杯。

時間複雜度為 $O(query\_row^2)$，空間複雜度同樣為 $O(query\_row^2)$。
