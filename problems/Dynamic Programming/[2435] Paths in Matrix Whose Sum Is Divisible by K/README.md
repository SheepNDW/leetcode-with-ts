## 2435. Paths in Matrix Whose Sum Is Divisible by K

這是一個常規的動態規劃問題。因為最終要求的路徑和需要是 k 的倍數，所以我們要考慮目前路徑和 mod k 的狀態。

我們定義 `dp[i][j][r]` 為從起點 `(0,0)` 到達位置 `(i,j)`，且路徑和 mod k 等於 `r` 的路徑數量。然後根據位置 `(i,j)` 的前一個位置 `(i-1,j)` 和 `(i,j-1)` 來轉移狀態：

```ts
dp[i][j][r] = dp[i-1][j][t] + dp[i][j-1][t]
```

其中 `t` 必須滿足 `(t + grid[i][j]) % k == r`。

第一列和第一欄因為只能從一個方向走來（第一欄只能往下，第一列只能往右），所以每個格子只有一條固定的路徑能到達。因此我們需要累加路徑和，並將相對應的 (路徑和 % k) 那個餘數位置設為 1。

最後答案回傳 `dp[m-1][n-1][0]` 即可。
