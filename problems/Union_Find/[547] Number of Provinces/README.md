## 547. Number of Provinces

給定一個 `n x n` 的矩陣 `isConnected`，其中 `isConnected[i][j] = 1` 表示第 `i` 個城市和第 `j` 個城市直接相連，否則為 `0`。求省份的數量。

#### 思路：

使用並查集 (Union-Find) 來解決此問題。

- 每個城市初始時都是自己的父節點
- 遍歷連接矩陣，將相連的城市進行合併
- 最後統計不同根節點的數量，即為省份數量

#### 實作細節

1. **初始化父節點陣列**：每個城市的父節點初始為自己
   ```typescript
   const Father = Array.from({ length: n }, (_, i) => i);
   ```

2. **FindFather (路徑壓縮)**：查找根節點並進行路徑壓縮優化
   - 遞迴查找根節點
   - 將路徑上的所有節點直接連接到根節點，減少後續查詢時間

3. **Union (合併)**：將兩個城市所屬的集合合併
   - 找到兩個城市的根節點
   - 將較大的根節點連接到較小的根節點

4. **統計省份數量**：使用 Set 統計不同根節點的數量
