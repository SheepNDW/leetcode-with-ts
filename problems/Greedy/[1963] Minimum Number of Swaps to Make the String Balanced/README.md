## 1963. Minimum Number of Swaps to Make the String Balanced

本題關鍵是要想到如果把所有括號隊都消去的話，剩餘的括號必然是像 `]]][[[` 的模式，這些位於兩邊的括號是無論如何都無法在原始字串中被配對的。

對於連續 n 個右括號 + 連續 n 個左括號，需要多少次 swap 才能夠讓他們彼此配對呢？可以試著列出前幾個例子找規律：

```
][: => [] 1 swap
]][[: => [][] 1 swap
]]][[[: => []][[] => [][][] 2 swaps
]]]][[[[: => []]][[[] => [][][][] 2 swaps
```

可以猜出規律是 `(n + 1) / 2` 次交換。
