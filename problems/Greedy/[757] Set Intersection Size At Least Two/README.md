## 757. Set Intersection Size At Least Two

整體思路類似於 435 的射氣球，我們想像如果我們能找出一系列 non-overlapping 的區間，那麼我們的答案必然需要在每個區間內至少採集兩點，我們採用 sort by end point 的方式去實作。

如果我們將區間按照後邊界從小到大排序後（相同後邊界時，短的區間在前，因為短區間能被覆蓋的話，相同後邊界的長區間也能被覆蓋），先考慮第一個區間，將它的後兩個點當作 `a` 和 `b`，然後初始答案為 2。

接下來對於每一個區間，我們考慮它的前邊界 `start`：

- 如果 `start` 小於等於 `a`，代表這個區間已經被前一個區間的兩個點覆蓋了，我們不需要做任何事，繼續往下處理下一個區間。
- 如果 `start` 大於 `b`，表示這個區間和前一個區間沒有重疊，我們需要再增加兩個點，將這個區間的後兩個點當作新的 `a` 和 `b`，並且答案加 2。
- 如果 `start` 小於等於 `b`，表示這個區間和前一個區間有重疊到一個點，我們只需要再增加一個點，把 `a` 點更新成 `b`，然後將這個區間的後一個點當作新的 `b`，答案加 1。

我們每次只關注選取的集合中的最大兩個點和目前區間 `[start, end]` 的關係，然後更新答案即可。
