## 3714. Longest Balanced Substring II

題目限制了只有三種字元，所以可以暴力分解為三種情況：

1. 只有一種字元的 substring，那麼整個 string 就是符合條件的。

2. 只有兩種字元的 substring，對於這部分的 substring，我們只需要對於前綴裡面兩種字元的頻次差用 hash 記錄。在遍歷 substring 時，如果發現目前的頻次差曾經出現過，那麼就意味著兩個前綴所夾的子串符合條件。注意，我們需要執行三次這樣的操作，分別對於 'a' 和 'b'、'a' 和 'c'、'b' 和 'c'。

3. 只有三種字元的 substring，其實就是針對整個原字串。類似於第二種情況，我們對前綴中的三種字元的頻次差 `{a-c, b-c}` 用 hash 記錄，遍歷 substring 時，如果發現目前的頻次差曾經出現過，就意味著兩個前綴所夾的子串裡 a, b, c 的頻次相同。
